# 日语歌曲学习 Web 版

- 通过 LRCLIB 搜索日语歌词
- 保存歌曲到浏览器本地
- 在歌曲详情页绑定本地音频，让音频与当前歌词关联
- 按歌词逐句学习
- 浏览器日语朗读
- 本地词典与语法点提示
- 生词收藏
- 整句机器翻译缓存
- 生成当前歌词句子的解析 Prompt
- 本地数据导出、导入、清空。音频文件本体保存在浏览器本地，不写入 JSON 导出文件

## 运行

```sh
npm install
npm run dev
```

分句学习页会生成当前歌词句子的解析 Prompt，可复制到你常用的 AI 工具里使用。

## 构建

```sh
npm run build
```
